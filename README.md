# Reconhecimento de Vínculo Empregatício — Landing Page

Página estática (`index.html`) para captação de clientes do escritório Medeiros & Costa Advocacia, voltada a trabalhadores contratados como PJ ou sem registro em carteira (CTPS) que podem ter direito ao reconhecimento de vínculo empregatício.

## Conteúdo

- Explicação em linguagem simples sobre reconhecimento de vínculo (CLT, Art. 3º) e seus elementos (subordinação, pessoalidade, habitualidade, onerosidade)
- Sinais de que a relação pode configurar vínculo empregatício, mesmo com contrato de PJ
- Chamadas para contato via WhatsApp e telefone
- Perguntas frequentes

## Tecnologia

Página única em HTML/CSS puro, sem build step ou dependências. Fontes carregadas via Google Fonts (Fraunces, Public Sans, Roboto Mono). Mesmo design system usado em outras landing pages do escritório (ex.: [wilson-pinto-imoveis](https://github.com/medeirosedu021-creator/wilson-pinto-imoveis)).

## Como visualizar localmente

Abra `index.html` diretamente no navegador, ou sirva a pasta com qualquer servidor estático, por exemplo:

```
npx serve .
```
